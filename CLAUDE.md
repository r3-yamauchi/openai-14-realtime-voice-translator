## 概要

これは OpenAI Realtime API と OpenAI Agents SDK を使用した**日本語→英語専用リアルタイム音声翻訳アプリケーション**です。従来の音声チャットとは異なり、AI が自発的な発言を一切行わず、ユーザーの日本語音声入力を即座に英語音声に翻訳する特化型システムとして実装されています。最新のリファクタリングにより型安全性、パフォーマンス最適化、保守性が大幅に向上しています。

## 最新の機能追加と改善

### 翻訳専用機能（重要な仕様変更）
- **完全受動的動作**: AI は挨拶や質問を一切行わず、日本語入力があるまで完全に沈黙
- **忠実な翻訳のみ**: 解釈、意見、追加情報を加えず、原文の意味を正確に英語に翻訳
- **初期挨拶の完全削除**: 「Hello. How can I help you today?」等の自発的発言を技術的に防止
- **interrupt()機能活用**: セッション接続直後に自動的に初期応答を中断

### 音声カスタマイゼーション機能
- **8種類の音声選択**: Alloy（デフォルト）、Ash、Ballad、Coral、Echo、Sage、Shimmer、Verse
- **動的音声変更**: 接続中でも音声タイプ変更時に自動再接続
- **5段階の音声速度調整**: とても遅い、遅い、普通、速い、とても速い
- **一元的なデフォルト管理**: `DEFAULT_VOICE_ID`定数による設定の一元化
- **設定永続化**: ローカルストレージでの音声設定・速度設定の保存

### UI/UX改善
- **翻訳特化のラベル**: 「トランスクリプト」→「翻訳履歴」
- **アプリタイトル変更**: 「日本語→英語 音声翻訳」
- **音声設定の統合表示**: 音声タイプと説明を同時表示（例：「Alloy - 汎用的で自然な音声」）
- **直感的なレイアウト**: 画面右上に音声速度・音声タイプ選択を配置

## アーキテクチャ

### 高レベルアーキテクチャ

この翻訳アプリケーションは以下の主要な技術スタックで構成されています：

- **フロントエンド**: Next.js 15 (App Router) + React 19 + TypeScript + Tailwind CSS

- **状態管理**: React Context + カスタムフック
- **音声処理**: MediaRecorder API + AudioContext + VAD（Voice Activity Detection）

### エージェント構成（翻訳専用）

- **翻訳専用設計**: 単一の翻訳エージェント (`simpleChat`) による日本語→英語変換

### データフロー

1. **音声入力**: ユーザーの日本語音声 → VAD検出 → 音声バッファ
2. **リアルタイム処理**: WebRTCストリーム → OpenAI Realtime API → エージェント処理
3. **翻訳生成**: 日本語音声認識 → 翻訳処理 → 英語音声合成
4. **音声出力**: 英語音声ストリーム → AudioElement → スピーカー出力

## 翻訳専用機能の実装詳細

### 技術的な実装

```typescript
// 音声設定とエージェント生成
export function createSimpleChatAgent(
  speechSpeed: SpeechSpeedLevel = 'normal', 
  voiceId: VoiceId = DEFAULT_VOICE_ID
): RealtimeAgent {
  return new RealtimeAgent({
    name: 'simpleChat',
    voice: voiceId, // 選択された音声ID
    instructions: getSpeechSpeedInstruction(speechSpeed),
    tools: [],
    handoffs: [],
  });
}

// 翻訳専用インストラクション生成
function getSpeechSpeedInstruction(speed: SpeechSpeedLevel): string {
  const baseInstruction = `あなたは日本語から英語への翻訳専門AIです。以下の規則を厳格に守ってください：

1. ユーザーが日本語で話した内容のみを英語に翻訳して返答する
2. 挨拶、質問、提案、コメントなど一切の自発的発言を禁止
3. 解釈、意見、追加情報、説明を一切加えない
4. 翻訳以外の応答を絶対に行わない
5. セッション開始時も含めて、こちらから話しかけない
6. ユーザーが日本語で話すまで完全に沈黙を保つ
7. 翻訳のみを忠実に、自然で流暢な英語で提供する

IMPORTANT: DO NOT speak first. DO NOT provide any initial greeting. WAIT for the user to speak in Japanese before responding. Only translate Japanese to English, nothing else.`;
  
  return `${baseInstruction} ${speedInstructions[speed]}`;
}
```
